# Text-to-Audio-Converter
Text to Audio Converter is a Python based application that converts written text into natural sounding speech across multiple languages. The project supports audio generation in MP3 format and demonstrates practical implementation of text to speech technology using Python libraries

# Text to Audio Converter

A Python-based Text to Speech project that converts user text into audio using multiple languages.

## Features

- Convert text to speech
- Multi-language support
- Save audio as MP3
- Simple command-line interface

## Technologies Used

- Python
- gTTS
- playsound

## Run Project

```bash
audio.py
```
